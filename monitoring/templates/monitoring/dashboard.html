Tips Menyimpan Data

Panduan lengkap menyimpan data denyut jantung dan suhu tubuh

01
Tempelkan Jari

Tempelkan jari ke sensor MAX30102 dan tunggu hingga nilai BPM stabil.

02
Nilai Terkunci

Lepas jari - nilai BPM akan terkunci untuk memudahkan pembacaan suhu.

03
Arahkan Sensor

Arahkan MLX90614 ke dahi (5 cm), tahan 3-5 detik hingga stabil.

04
Simpan Data

Klik "Simpan Data" untuk menyimpan hasil pengukuran ke database.

Catatan: Anda bisa menggunakan sensor tanpa menyimpan data - cukup tempelkan jari dan arahkan sensor ke dahi.
